# Walkthrough - Teacher Content Creation Flow

## Managing Subjects
1. Log in as a **Teacher** or **Admin**.
2. Go to the **Subjects** page.
3. Click **"Thêm môn học"** (Add Subject) to create a new category.
4. Provide a name and icon.

## Creating a Curriculum
1. Within a Subject, click **"Tạo giáo trình mới"** (Create New Curriculum).
2. Enter details like grade level and publisher.
3. Upload a cover image or configuration file.
4. Save the curriculum to begin adding lessons.

## Authoring a Lesson
1. Select a Curriculum and click **"Thêm bài học"** (Add Lesson).
2. **Step 1**: Enter the title and basic description.
3. **Step 2**: Write the structured lesson content with rich text.
4. **Step 3**: Use the **"Import Quiz/Flashcards"** button to batch-create interactive items.
5. **Step 4**: Preview the lesson to verify layout and formatting.
6. Click **"Hoàn tất"** (Finish) to publish the lesson.

## User Administration (Admin Only)
1. Go to the **Admin** dashboard.
2. View the full list of users.
3. Use the **"Tạo tài khoản"** button to manually register new teachers or students.
4. Reset passwords or delete accounts as needed.
